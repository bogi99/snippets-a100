# Generic Dev Container

A reusable development container configuration for consistent development environments across different projects.

## Features

- Pre-configured development environment
- Containerized setup for reproducibility
- Easy to customize and extend
- Works with VS Code Dev Containers

## Usage

1. Clone this repository
2. Open in VS Code with Dev Containers extension
3. VS Code will automatically build and connect to the container

## Requirements

- Docker
- Visual Studio Code
- Dev Containers extension

## Customization

Modify the `.devcontainer/devcontainer.json` file to add tools, extensions, or change settings according to your project needs.
